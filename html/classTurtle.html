void Turtle::restoreState ( )
inline

Pops the last saved stack from the stack and sets the Turtle to that state for future line drawing. Outputs an error message in case the stack is empty.

void Turtle::saveState ( )
inline

Part of the language of many L-systems involve back-tracking to a state. Here we can save the current state of the Turtle (current co-ordinate and rotation) so we can move back to it later. States are stored in an std::stack of tuples.
